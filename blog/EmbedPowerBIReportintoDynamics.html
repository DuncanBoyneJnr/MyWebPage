How to Embed a Power BI Report into Dynamics

Step 1: Get Your Power BI Report Ready

  1. Make sure your report is published to a Power BI workspace that’s linked to your Dynamics environment.
  2. Copy the Report ID and Workspace ID:
    • Open your report in Power BI.
    • Look at the URL in your browser. You’ll see something like this: 
      https://app.powerbi.com/groups/WorkspaceID/reports/ReportID
    • Note down the Workspace ID and Report ID.

Step 2: Add the Power BI Report to Dynamics

  1. Go to Dynamics:
    • Log into Dynamics and navigate to the area where you want the report to appear.
  2. Edit the Form:
    • Go to the specific table (e.g., “Accounts” or “Opportunities”).
    • Click Settings > Advanced Settings > Customization > Customize the System.
    • Find and open the form where you want the report embedded.
  3. Add an IFRAME for Power BI:
    • In the form editor, click Insert > IFRAME.
    • Name the IFRAME (e.g., PowerBI_Report).
    • Use this URL format for the IFRAME source: 
      https://app.powerbi.com/reportEmbed?reportId=<ReportID>&groupId=<WorkspaceID>
    • Replace <ReportID> and <WorkspaceID> with the ones you noted earlier.
    • Check the boxes for "Pass record object-type code and unique identifier as parameters" (if needed).

Step 3: Set Permissions

  1. Ensure User Access:
    • Users need Power BI licenses to view the embedded report.
    • Make sure they have access to the workspace in Power BI.

Step 4: Test and Publish

  1. Save and Publish the Form.
  2. Go to Dynamics, navigate to the form, and check if the report appears as expected.
